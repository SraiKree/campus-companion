# Student Courses Page

This page combines the Assignments and Grades functionality into a single unified interface with tabs.

## Features

### Assignments Tab
- View all assignments with filtering by status (pending, submitted, graded) and subject
- Submit assignments with file upload (max 15MB)
- View assignment details including deadline, subject, and description
- Track submission status with color-coded badges
- View grades once assignments are graded

### Grades Tab
- View academic performance statistics (CGPA, Total Credits, Semesters, Current GPA)
- Select different semesters to view grades
- View subject-wise grades with progress bars
- See detailed grade breakdown including:
  - Grade letter and percentage
  - Obtained marks vs total marks
  - Number of assignments per subject
  - Credits per subject

## Design
The page follows the Figma design specifications with:
- Clean tab-based navigation
- Consistent color scheme (#e05252 for primary actions, #f2f0ed for backgrounds)
- Rounded corners and modern card layouts
- Responsive grid layouts
- Status badges with appropriate colors

## Navigation
Accessible from the main student navigation sidebar under "Courses"
