# Attendance System User Manual

## Overview

The Attendance System provides a comprehensive solution for tracking student attendance across different classes, academic periods, and semesters. It connects faculty and students through an intuitive interface that makes attendance management efficient and transparent.

## System Architecture

### Academic Structure
- **Academic Years**: I, II, III, IV (for undergraduate), I MBA, II MBA (for postgraduate)
- **Semesters**: I through VIII (depending on the program)
- **Departments**: CSE, CSM, CSD, AERO, IT, MECH, EEE, CSIT, ECE, MBA
- **Sections**: A, B, C, etc. (within each department and year)

### Class Organization
- Faculty create classes through the **Timetable System**
- Each class is associated with:
  - Subject name and code
  - Department and section
  - Academic year and semester
  - Day of the week and period timings
  - Room number (optional)
  - Lab designation (if applicable)

## For Faculty Members

### Getting Started

1. **Login** using your faculty credentials
2. Navigate to the **Faculty Dashboard**
3. Click on the **Attendance** tab

### Prerequisites

Before marking attendance, ensure you have:
- Created classes in the **Timetable** section
- Selected the appropriate **Academic Period** (year and semester)

### Marking Attendance

#### Step 1: Select Academic Period
- Use the dropdown to choose the academic year and semester
- The system shows all available periods based on your created classes

#### Step 2: Choose a Class
- View all your classes in a card-based layout
- Each class card displays:
  - Subject code and name
  - Department and section (e.g., "CSE-A")
  - Day of the week
  - Period timing
  - Room number
  - Lab indicator (if applicable)

#### Step 3: Mark Student Attendance
Once you select a class, you'll see:

**Class Information Header**
- Subject details
- Department, section, and academic period
- Day and period information

**Student Management Tools**
- **Search Bar**: Find students by name or roll number instantly
- **Sort Options**: 
  - Sort by Name (A-Z or Z-A)
  - Sort by Roll Number (ascending/descending)
- **Pagination**: Navigate through large class lists (10, 20, 50, or 100 students per page)

**Attendance Controls**
- **Individual Marking**: Check/uncheck each student's attendance
- **Bulk Actions**:
  - "Mark All Present" - Sets all students as present
  - "Mark All Absent" - Sets all students as absent
- **Real-time Summary**: See present/absent counts as you mark

**Student List Features**
- Each student shows:
  - Full name
  - Roll number
  - Present/Absent status badge
  - Checkbox for attendance marking

#### Step 4: Save Attendance
- Click **"Save Attendance"** to record the session
- The system automatically timestamps the attendance
- You'll receive confirmation when saved successfully

### Advanced Features

#### Large Class Management
- **Search Functionality**: Type student names or roll numbers for instant filtering
- **Smart Pagination**: Navigate through hundreds of students efficiently
- **Sorting**: Organize students alphabetically or by roll number
- **Filtered Counts**: See how many students match your search criteria

#### Academic Period Tracking
- Attendance is stored with academic year and semester context
- Historical attendance records are maintained
- Easy switching between different academic periods

## For Students

### Accessing Your Attendance

1. **Login** using your roll number as both username and password
2. Navigate to the **Student Dashboard**
3. Click on the **Attendance** tab

### Understanding Your Attendance Display

#### Subject-wise Attendance Cards
Each subject you're enrolled in shows:
- **Subject Name and Code**
- **Faculty Name** (who teaches the subject)
- **Department and Section**
- **Academic Period** (year and semester)

#### Attendance Statistics
For each subject, you can see:
- **Total Classes Conducted**
- **Classes Attended**
- **Attendance Percentage**
- **Visual Progress Bar** showing your attendance level

#### Color-coded Status
- **Green**: Good attendance (typically 75% and above)
- **Yellow/Orange**: Warning level (between 65-75%)
- **Red**: Critical level (below 65%)

### Attendance Calculation

Your attendance percentage is calculated as:
```
Attendance % = (Classes Attended / Total Classes Conducted) × 100
```

### Academic Period Context

- Attendance is tracked separately for each academic year and semester
- You can view historical attendance from previous semesters
- Current semester attendance is prominently displayed

### Understanding Attendance Requirements

Most institutions require:
- **Minimum 75% attendance** for exam eligibility
- Some subjects may have different requirements
- Lab sessions and theory classes are tracked separately

## Troubleshooting

### Common Issues

**Faculty Cannot See Students**
- Verify the correct academic period is selected
- Ensure the class was created in the timetable system
- Check that students are enrolled in the correct department/section

**Students Not Seeing Attendance**
- Attendance appears only after faculty marks and saves it
- Verify you're checking the correct academic period
- Ensure you're enrolled in the subject

**Search Not Working**
- Try partial names or roll numbers
- Check spelling and spacing
- Use the sort function as an alternative

**Pagination Issues**
- Adjust items per page setting
- Use search to narrow down results
- Check total student count in the class
